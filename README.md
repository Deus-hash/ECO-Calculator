# ECO.Calculator

**Экология Конструкций Ограждения** — инженерный калькулятор для расчёта теплотехнических и экологических параметров наружных стен.

Разработан для проектировщиков, строителей и экологов. Позволяет оценить массу, углеродный след и воплощённую энергию многослойных ограждающих конструкций, а также рассчитать теплопотери и расход газа на отопление.

---

## Возможности

### Расчёт
- Масса 1 м² и общая масса стен
- CO₂ выбросы (кгCO₂экв/м²) — общие и на единицу площади
- Воплощённая энергия (МДж/м²) — общая и на единицу площади
- Условное сопротивление теплопередаче R₀
- Теплопотери Qст.год и Qгод общие
- Расход газа V₁ и Vобщ

### База материалов
- Кирпич керамический и силикатный
- Газосиликатные, керамзитобетонные блоки
- Минеральная вата, пенополистирол, ЭППС
- Цементно-песчаный раствор
- Арматурные сетки (сталь, стеклопластик, базальт, углепластик)
- Все характеристики: плотность, теплопроводность, F-фактор, e-фактор
- Фильтрация и поиск по названию

### Режимы работы
- **Обычный режим** — значения округляются для удобного отображения
- **Инженерный режим** — полная точность без округлений

### Интерфейс
- Светлая и тёмная темы (сохраняется в localStorage)
- Добавление / удаление слоёв стены
- Учёт оконных и дверных проёмов
- Выбор типа армирования и его массы

### Экспорт
- **PDF** — открывает окно печати браузера (можно сохранить в PDF)
- **XLSX** — скачивание отчёта в формате Excel
- **DOCX** — скачивание отчёта в формате Word
- Исходные данные, конструкция стены и все результаты

## Установка и запуск

### Локальный (фронт без сервера)

Просто откройте файл в браузере:

```
web/index.html
```

### С сервером (Python)

```bash
pip install -r requirements.txt
python server.py
```

Затем откройте в браузере: [http://localhost:3000](http://localhost:3000)

### Структура проекта

```
ECO-Calculator/
├── index.html          # Главная страница (с API-запросами к серверу)
├── script.js           # Фронтенд-логика (расчёт, слои, экспорт)
├── style.css           # Стили и темы
├── server.py           # Запуск FastAPI-сервера
├── main.py             # FastAPI-приложение с эндпоинтами
├── materials_db_web.py # База данных материалов
├── img/                # Изображения
├── web/                # Автономная версия (без сервера)
│   ├── index.html
│   ├── script.js
│   └── style.css
├── requirements.txt
└── README.md
```
