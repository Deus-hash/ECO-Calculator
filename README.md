# ECO.Calculator

Инженерный калькулятор для расчёта теплотехнических и экологических параметров наружных стен.

## Возможности

- Расчёт массы, CO₂ выбросов и воплощённой энергии многослойной стены
- Автоматический расчёт R₀, теплопотерь, расхода газа
- База материалов с возможностью фильтрации и поиска
- Учёт армирования (металлическая сетка, стеклопластиковая, базальтовая, углепластиковая)
- Учёт оконных и дверных проёмов
- Обычный и Инженерный режимы
- Светлая и тёмная темы
- Экспорт отчёта в PDF, XLSX, DOCX

## Запуск

### Локальный (фронт без сервера)

Откройте `web/index.html` в браузере.

### С сервером (Python)

```bash
pip install -r requirements.txt
python server.py
```

Откройте `http://localhost:3000` в браузере.
